# HNCGAT 论文创新点分析

## 🎯 论文的创新点（不是 Table 3）

### 主要创新点

1. **HNCGAT 模型本身**
   - **Heterogeneous Neighbor Contrastive Graph Attention Network**
   - 结合了异质图注意力网络和对比学习

2. **异质邻居对比学习框架**（核心创新）
   - 设计了一种新的对比学习损失函数
   - 将异质网络拓扑结构作为监督信号
   - 定义正负样本对来指导节点嵌入学习

3. **首次应用于植物 MPI 预测**
   - 首次将异质图神经网络应用于植物代谢物-蛋白质相互作用预测
   - 构建了第一个 Arabidopsis 异质网络数据集

4. **多模态信息融合**
   - 整合了蛋白质、代谢物、功能注释三种节点类型
   - 整合了五种边类型（MP, MM, MF, PP, PF）

## 📊 Table 3 的作用（不是创新点）

### Table 3 是**性能评估实验**，用来：

1. **验证模型性能**
   - 测试模型在不同训练比例下的表现（30%, 50%, 90%）
   - 证明模型在不同数据量下都能工作

2. **技术细节测试**
   - 测试不同的边操作符（Concatenate, L1, L2, Hadamard, Average）
   - 这是**实现细节**，不是创新点
   - 用来找到最佳配置

3. **与 Baseline 对比**
   - 与 GCN, HGT, HAN 等方法对比
   - 证明 HNCGAT **优于**现有方法

4. **鲁棒性验证**
   - 5次随机划分，报告平均值和标准差
   - 证明结果稳定可靠
