# 训练比例（Training Ratio）说明

## 🔍 什么是"训练比例"？

### 训练比例 = 数据划分比例

**不是模型本身，而是数据划分方式**：

```
总数据集 (100%)
├── 训练集 (90%) ← 用来训练模型
└── 测试集 (10%) ← 用来评估模型性能
```

### 论文中的实验设计

**作者测试了三种训练比例**：
- **30%训练比例**：30%数据训练，70%数据测试
- **50%训练比例**：50%数据训练，50%数据测试
- **90%训练比例**：90%数据训练，10%数据测试

**目的**：验证模型在不同数据量下的表现

## 📊 论文中的结果

### Table 3 展示了不同训练比例下的结果

| 训练比例 | 目的 | 说明 |
|---------|------|------|
| 30% | 小样本测试 | 验证模型在数据少时是否还能工作 |
| 50% | 中等样本测试 | 验证模型在中等数据量下的表现 |
| 90% | 大样本测试 | 验证模型在数据充足时的最佳表现 |

**所有实验都使用同一个HNCGAT模型**，只是训练数据量不同。
